# Superstore Sales Dashboard

[![Power BI](https://img.shields.io/badge/Power%20BI-Desktop-blue)](https://powerbi.microsoft.com/)

## Overview
Interactive **Power BI dashboard** to analyze Superstore sales data. Visualizations help track sales trends, product performance, and regional distribution, enabling data-driven decisions.

---

## Features
- **Map**: Regional sales distribution  
- **Pie & Donut Charts**: Category & sub-category sales breakdown  
- **Funnel Chart**: Sales pipeline overview  
- **Line Chart**: Sales trends over time  
- **Card Charts**: Key metrics like total sales, profit, and orders  

---

## Dataset
- **Source**: Superstore Sales Dataset  
- **Fields**: Order ID, Product, Category, Sub-Category, Sales, Profit, Region, Order Date, etc.  

---

## How to Use
1. Open the `.pbix` file in **Power BI Desktop**  
2. Use **filters & slicers** to explore data by region, category, or time period  
3. Hover over visualizations for detailed insights  
4. Analyze trends and performance interactively
